## Preparación

- [ ] La persona no redactó el procedimiento.
- [ ] Recibió solo el material disponible durante el trabajo.
- [ ] El ambiente usa datos de demostración.
- [ ] Se explicó que se evalúan las instrucciones, no a la persona.
- [ ] El observador sabe cuándo detener el ejercicio por seguridad.
- [ ] La sesión puede repetirse desde un estado conocido.

## Preguntas posteriores

Pregunte después de la práctica, sin sugerir la respuesta:

1. ¿Qué parte fue más fácil?
2. ¿En qué parte dudó?
3. ¿Qué palabra o símbolo no quedó claro?
4. ¿Qué esperaba ver y no apareció?
5. ¿Cuándo se detendría para pedir ayuda?
6. ¿Qué información daría al pedir ayuda?
