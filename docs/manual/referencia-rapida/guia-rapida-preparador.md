# Guía rápida del preparador

**Esta hoja sirve como recordatorio. La capacitación y el manual completo siguen
siendo necesarios.**

## Preparar un pedido

1. Escanee su gafete.
2. Introduzca su clave y pulse `A`.
3. En `READY`, pulse `A` para recibir una tarea.
4. Vaya a la ubicación mostrada en `GO TO` y escanéela.
5. Recoja el artículo mostrado en `PICK` y escanéelo.
6. Cuente las unidades.
7. Confirme con `A` únicamente si la cantidad coincide.
8. Espere `PICK OK ✓`.
9. Compruebe que aparezca `synced ✓` o espere a que desaparezca `Q:`.

## Si algo no coincide

| Situación | Acción |
|---|---|
| Ubicación incorrecta | No recoja nada; vaya a la ubicación indicada |
| Artículo incorrecto | Devuélvalo y busque el artículo indicado |
| Cantidad diferente | Pulse `B`, vuelva a contar y avise al supervisor si continúa diferente |
| `NO TASKS` | Pulse `A` para consultar otra vez o avise al supervisor |
| `OFF` antes de una tarea | Busque cobertura; no puede tomar una tarea nueva |
| `OFF` durante una tarea | Termine la tarea, mantenga encendida la terminal y busque cobertura |
| `Q:` con un número | Hay información pendiente; no cierre la sesión ni apague la terminal |
| `SYNC FAILED` | No repita el trabajo; anote la tarea y el código, y avise al supervisor |

## Antes de terminar el turno

- Compruebe que la pantalla muestre `ON`.
- Compruebe que no aparezca `Q:`.
- Mantenga pulsado `START` para cerrar la sesión.
- Espere `SCAN BADGE`.
- Entregue la terminal para carga.

## Nunca haga esto

- No comparta su gafete o clave.
- No confirme sin contar.
- No sustituya un artículo por otro parecido.
- No repita físicamente una preparación con `SYNC FAILED`.
- No apague la terminal mientras aparezca `Q:`.
- No comunique su clave al informar un problema.

**Al pedir ayuda, indique:** usuario, terminal, tarea, ubicación, artículo, mensaje,
hora aproximada y estado `ON`/`OFF`/`Q:`.
